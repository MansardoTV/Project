# Проектирование и реализация масштабируемого конвейера обработки больших данных для анализа отзывов ресторанов

## Описание
Наш проект это парсер для сбора отзывов с ресторанов и кафе с Яндекс.Карт. Парсер анализирует тональность отзывов и разделяет их на позитивные, негативные и нейтральные с помощью анализа ключевых слов(меток с массива) и оценок. 

## Цель проекта
Собрать данные о различных заведениях (ресторанах, кафе) для создания набора данных, который может быть использован для анализа пользователями при выборе лучшего места для перкуса.

## Как использовать
1. Установите все необходимые зависимости, выполнив `pip install -r requirements.txt`.
2. Запустите скрипт `parser_1.py` передав необходимые параметры (например, URL или координаты) для определения области или объектов для парсинга.
3. Парсер получит доступ к страницам Яндекс.Карт, используя указанные параметры, и соберет информацию такую как: координаты, оценку, категории, количество отзывов, адрес, время работы и комментарии пользователей для каждого объекта.
4. Собранные данные будут сохранены в формате JSON в n размере для каждого ресторана отдельно и один json файл с полной статистикой.

## Структура данных
Для каждого обработанного объекта собираются следующие данные:
-Координаты
-Оценка
-Категории
-Количество отзывов
-Адрес
-Время работы
-Показатель физического лица
-Показатель юридического лица
-Комментарии пользователей
-Анализ тональности отзывов (позитивные, негативные, нейтральные)
